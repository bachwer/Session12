Spring Boot quản lý các cấu hình mặc định (Default Properties) thông qua cơ chế Spring Boot Auto Configuration. Khi ứng dụng khởi động, Spring Boot sẽ đọc các giá trị mặc định từ:

SpringApplication: lớp khởi tạo chính, nó tự động gán các giá trị mặc định (ví dụ: port = 8080).

application.properties hoặc application.yml: nơi người dùng có thể ghi đè các giá trị mặc định.

Spring Boot Reference Documentation: toàn bộ danh sách các cấu hình có thể tìm thấy trong tài liệu chính thức của Spring Boot (Spring Boot Properties list).
